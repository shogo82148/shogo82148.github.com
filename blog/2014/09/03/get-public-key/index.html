秘密鍵から公開鍵をつくる

githubに公開鍵を登録しようと思ったけど、 手元に秘密鍵しかなくて困った時のメモ。

ssh-keygenを使うとできます。

# 秘密鍵を読み込んで公開鍵を出力する
ssh-keygen -y -f ~/.ssh/id_rsa

この公開鍵って登録したっけ？ ってときには以下のコマンドでフィンガープリントを確認できます。

# 公開鍵のフィンガープリントを取得する
ssh-keygen -l -f ~/.ssh/id_rsa.pub
