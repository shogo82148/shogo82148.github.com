AWS Lambda Perl Runtime Layer in 大阪リージョン を公開しました

AWS 大阪リージョンが一般利用可能になりました！

  • AWS Asia Pacific (Osaka) Region Now Open to All, with Three AZs and More Services
  • [AWS] 日本 2 番目となる大阪リージョン ap-northeast-3 が利用可能になりました
  • ［速報］「AWS 大阪リージョン」正式オープン。大阪ローカルリージョンを拡張し 3 つのアベイラビリティゾーンから構成、事前申し込みなど不要に

というわけで、 AWS Lambda Perl Runtime AWS::Lambda in Osaka を公開しました。
