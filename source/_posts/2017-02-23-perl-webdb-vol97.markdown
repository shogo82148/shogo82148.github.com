# 余談
ところで、**Vol.97**と**第43回**ってどっちも**素数**ですね！
雑なプログラムを書いて調べてみたところ、
両方素数になるのはVol.83, 第29回以来、**7回目**(これも**素数**だ！)。
次はVol.101, 第47回です。
そのときのPerl Hackerは誰になるのでしょうか。楽しみですね！

``` perl
use warnings;
use strict;

sub is_prime {
    my $n = shift;
    return 0 if $n < 2;
    my $i = 2;
    while($i*$i<=$n) {
        return 0 if $n % $i == 0;
        $i++;
    }
    return 1;
}

my $i = 1;
for my $n(1..200) {
    my $m = $n-43+97;
    if (is_prime($n) && is_prime($m)) {
        printf "%3d: Vol.%3d, No.%3d\n", $i, $m, $n;
        $i++;
    }
}
```
